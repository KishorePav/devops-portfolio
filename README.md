# DevOps & Cloud Consulting Portfolio

Focused consulting site for Kishore Choudhury's independent DevOps practice.

## Positioning

This project intentionally remains separate from the broader `KishorePav.github.io` engineering profile. It presents DevOps, cloud delivery, Kubernetes and observability services without narrowing the owner's primary backend and platform engineering identity.

## Local preview

```bash
python3 -m http.server 8000
```

Open `http://localhost:8000`.

## Deployment

The `Deploy website` workflow validates links, uploads the static site and deploys it to GitHub Pages. In repository settings, set **Pages → Source** to **GitHub Actions**.

Expected URL:

```text
https://kishorepav.github.io/devops-portfolio/
```
